# 🎨 Tudo Sobre Design - Web

### 📝 O que é o projeto?
O **Tudo Sobre Design** é um curso fictício de Design Gráfico desenvolvido para fins de estudo. O projeto simula a jornada completa de um aluno dentro de um ecossistema de vendas:
1. **Landing Page:** Página principal onde o usuário conhece o curso, os benefícios e a autoridade da marca.
2. **Pagamento:** Fluxo simulado via Kiwify.
3. **Tela de Agradecimento:** Após o checkout, o usuário é direcionado para uma página de sucesso com instruções.
4. **Plataforma de Alunos:** O aluno é encaminhado para a área de login onde, utilizando o e-mail cadastrado, define sua senha e acessa o conteúdo exclusivo.

---

### 🎨 Design e Prototipagem (Figma)
Todo o projeto foi planejado visualmente no **Figma** antes de iniciar o desenvolvimento do código. O foco foi criar uma interface moderna, limpa e com alta taxa de conversão (UI/UX).

### 💡 Observações de Aprendizado
Este foi o meu **primeiro projeto web completo**. Durante o processo, pude colocar em prática conceitos fundamentais de desenvolvimento front-end. 

Como ainda estou aprimorando meus conhecimentos em **JavaScript**, utilizei o auxílio de IA para a implementação de funcionalidades mais complexas, como:
